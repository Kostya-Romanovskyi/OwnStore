Project 'ownCinema'

Implemented a web application where users can find their favorite movies or new movies.

User`s can search movies by categorys, by handle search and by according to the preferences
that the web application offers on the main page of the movie

If user want, he can authorized with Google and open library for himself, 
where he can add his favorite movies or movies which he want to watch later

In project using technologys like :
HTML, CSS, React, JavaScript, Rest API, AJAX, adaptive layout, Redux, Vite.

For authorization from Google, and save userData using FireBase DB.
